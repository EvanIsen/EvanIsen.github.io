Description

Ce projet a été réalisé dans le cadre de ma première année dans le bachelor, c'était un projet en équipe de 3 sur Unity dans lequel on devait modéliser notre campus. Cela avait pour but de valider nos débuts dans le milieu du jeu vidéo. J'ai modélisé quelques assets, comme les tables ou le canapé. J'ai aussi géré l'intégration de toutes les assets sur Unity, J'ai fait la cuisson des lumières. Le système pour ouvrir les portes est aussi de moi, basé sur un trigger et une rotation de l'objet via C#.
